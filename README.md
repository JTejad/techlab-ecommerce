# TechLab E-commerce

Proyecto final del curso Backend Java.

La idea del proyecto es construir una API REST con Spring Boot y MySQL para gestionar un e-commerce, integrada con un frontend HTML, CSS y JavaScript.

## Estructura

- `backend/`: API REST desarrollada con Java, Spring Boot, Spring Data JPA y MySQL.
- `frontend/`: interfaz web para consumir la API, mostrar productos, manejar carrito y realizar pedidos.

## Funcionalidades esperadas

- gestion de productos
- gestion de categorias
- carrito de compras
- creacion de pedidos
- historial de pedidos
- actualizacion de stock
- manejo de errores y validaciones
